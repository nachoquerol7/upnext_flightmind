logo_plane.stl — plano 1×1 unidades (plano horizontal Z=0); el nodo escala con logo_scale_*.

Para usar tu logo: exporta STL/OBJ y pásalo con el parámetro ROS logo_mesh_uri
(p. ej. file:///ruta/absoluta/mi_logo.stl). Desactivar el plano por defecto: logo_mesh_enable:=false
o sustituir el fichero manteniendo el nombre.
